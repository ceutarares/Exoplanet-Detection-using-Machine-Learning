## Exoplanet Detection Using Machine Learning

In this project, we aim to detect exoplanets orbiting distant stars using machine learning techniques. The primary method employed throughout this work is the **transit method**, one of the most widely used approaches in modern astronomy for identifying exoplanets.

The transit method is based on observing periodic dips in a star’s brightness. When a planet passes in front of its host star (from the observer’s perspective), it blocks a small fraction of the star’s light, causing a temporary decrease in observed luminosity. These variations are recorded as light curves and can be analyzed to infer the presence of an exoplanet.

This study focuses particularly on large exoplanets similar in size to Jupiter, commonly referred to as **Hot Jupiters**. These planets orbit very close to their host stars, making their transits more frequent and easier to detect due to the more pronounced drop in brightness.

By analyzing stellar light curves and applying machine learning models, this project explores how signal processing and classification techniques can be used to distinguish between stars with and without orbiting exoplanets.
